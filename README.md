# 🚀 WebPilot Pro

**WebPilot Pro** is a modern all-in-one Chrome extension that combines SEO analysis, developer utilities, browser tools, and website inspection into a single, lightweight toolkit. Whether you're an SEO expert, web developer, digital marketer, QA engineer, or website owner, WebPilot Pro helps you analyze, inspect, and optimize websites faster.

---

## ✨ Features

- 🔍 On-Page SEO Analysis
- 📊 Analytics & Tracking Detection
- 🤖 AI Content Detection
- 🖼️ Image Extractor & Downloader
- 🔗 Internal & External Link Inspector
- 📋 Copy Selected Text
- 🍪 Cookies Inspector
- 💾 Local & Session Storage Viewer
- 🧹 Cache & Browser Data Cleaner
- 🛠️ Developer Utilities
- ⚡ Fast, Lightweight & Easy to Use

---

## 📦 Installation

1. Clone this repository.

```bash
git clone https://github.com/your-username/WebPilot-Pro.git
```

2. Open **Google Chrome**.
3. Go to **chrome://extensions/**.
4. Enable **Developer Mode**.
5. Click **Load unpacked**.
6. Select the project folder.
